# NEDTP  
## Installation  
### Requirements  
* The model could be run in the Python 3.7.4  
* numpy == 1.16.5
* sklearn == 0.19.2
* lightgbm == 2.2.3
* gensim == 3.8.0
## Run NEDTP through the provided features  
Execute in the command line:`python main.py`
